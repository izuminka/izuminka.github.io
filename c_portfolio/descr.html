ML PROPERTIES OF MATERIALS

This a demo project to show various ways of encoding structural and chemical information of a material into a "materials descriptor" which is used to train a machine learning model that associates structo-chemical information with a materials' property. Given a new materials descriptor one can use the model to predict the corresponding property. The project is primarily completed with Python 3 and several Bash scripts
